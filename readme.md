# flask + flask_rest_api + flask_sqlalchemy demo
简单的flask整合orm和rest接口的例子

## 依赖

flask的包
`pip install flask flask_restful flask_sqlalchemy`
wsgi服务相关
`pip install Werkzeug`
可能需要mysql连接驱动
`pip install mysql-connector-python`


## 文件列表

### 1. src/apps.py

APP默认配置

### 2. src/models.py

数据库配置

### 3. src/views.py

rest-api相关配置

### 4. main.py

主运行程序
